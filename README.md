# transelator_tif

功能包括`根据鸟瞰tif图切片成地图瓦片（OSGeo TMS to OSM）`及`地理配准`

- `根据鸟瞰tif图切片成地图瓦片` - `python cut_tiles.py`
- `地理配准` - `python tiff_translator/tif_translator.py`
